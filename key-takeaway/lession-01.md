# setup env
- `nvm` node version manager
- 

- install IDE Visual studio code
- install git
- install default terminal = git bash
- connect git -> github = sshkey
    - id_rsa: git, private
    - id_rsa.pub: set github, public
    - `ssh-keygen -t rsa -b 4096 -C "your@email"` comand generate ssh key
        - `-t` key type
        - `-b` bit lenght
        - `-C` comment

- `npm init playwright@latest` install required package for playwright & create structure project
